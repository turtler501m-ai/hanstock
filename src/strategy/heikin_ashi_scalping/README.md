# 알파 하이킨아시 스캘핑 전략

참고 영상: https://youtu.be/55hFmXD0lh0

## 영상 전략 분석 요약

영상은 하이킨아시를 일반 캔들의 노이즈를 줄여 추세를 보기 쉽게 만드는
차트로 설명하고, 실전 매매법 여섯 가지를 제시합니다.

1. 지지선/저항선 반전 매매
   하락 추세가 지지선 근처에서 약해지고 도지 또는 강한 양봉이 나오면
   반등 진입을 고려합니다.

2. 지지선 이탈 돌파 매매
   지지선을 강한 음봉으로 이탈하면 하락 추세 지속으로 보고 매도 진입을
   고려합니다.

3. EMA20 눌림목 매매
   가격이 EMA20 위에 있고 눌림목에서 도지가 연속으로 나온 뒤 상승
   하이킨아시가 확인되면 매수합니다.

4. EMA10/EMA20 추세 지속 매매
   EMA10이 EMA20을 상향 돌파하고, 눌림목에서 몸통 기준으로 EMA10을
   지킨 뒤 상승 캔들이 나오면 매수합니다.

5. RSI 50선 결합 매매
   하이킨아시 색상이 상승으로 바뀌면서 RSI가 50선을 상향 돌파하면
   매수합니다. 반대로 하락색 전환과 RSI 50선 하향 이탈은 매도 신호로
   봅니다.

6. 알파 하이킨아시 매매
   하이킨아시를 한 번 더 평균 처리해 잔파동을 줄입니다. 색상이 하락색에서
   상승색으로 바뀌면 매수하고, 3R 부근에서 절반 익절 후 색상이 다시
   하락색으로 바뀔 때 잔여 물량을 정리하는 방식입니다.

## 구현 규칙

이 구현은 영상의 마지막 핵심인 알파 하이킨아시 규칙을 중심으로 만들고,
EMA와 RSI 조건을 보조 필터로 사용합니다.

매수 점수 조건:

- 알파 하이킨아시가 하락색에서 상승색으로 전환
- 가격이 EMA20 위에 있고 EMA10이 EMA20 이상
- 도지 압축 이후 상승 방향 확정
- 일반 하이킨아시도 2연속 상승 캔들로 확인
- RSI가 50선을 상향 돌파하거나 50선 위 상승 모멘텀 유지
- 거래량이 20개 캔들 평균 대비 1.2배 이상

리스크 관리:

- 진입가는 최근 종가
- 손절가는 최근 알파 하이킨아시 스윙 저점
- 목표가는 2R, 3R 계산값 제공
- 대시보드 후보 점수는 최대 5점

## 대시보드 등록

대시보드의 커스텀 전략 로더는 `src/strategy/custom_rules/*.py` 파일을
스캔합니다. 그래서 실제 전략 본체는 이 폴더에 두고, 아래 래퍼 파일로
자동 등록되도록 구성했습니다.

```text
src/strategy/custom_rules/heikin_ashi_scalping_strategy.py
```

전략 ID:

```text
heikin_ashi_scalping_strategy
```
