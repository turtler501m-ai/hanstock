## 거래비용 모델 (핵심 신규)
trade_cost = 0.15% per round-trip (0.05% commission + 0.10% slippage)
gross_alpha = strategy_return - SPY_return
net_alpha = gross_alpha - trade_cost
모든 ranking 표에 gross / net 둘 다 병기.
비용 sensitivity 분석:
0% / 0.05% / 0.10% / 0.15% / 0.20% / 0.30% 6 수준
각 수준에서 skill별 net_alpha + 유의성 라벨.

## 통계 분석 (paper-grade 강화)
각 skill × horizon:
- gross/net 평균 alpha + cluster bootstrap 95% CI (ticker cluster, n=1000)
- Sharpe, MDD, win rate
- t-statistic + p-value (cluster robust SE)

## Bonferroni 다중비교 보정 (핵심 신규)
N_tests = 8 skill × 3 horizon = 24
α_adjusted = 0.05 / 24 = 0.00208
새 라벨:
- ci_low > 0 AND p < 0.00208 → "✅ Bonferroni 유의"
- ci_low > 0 (only) → "⚠️ 단일검정 유의 (다중비교 후 비유의)"
- 그외 → "❌ 비유의"

| ✅ Bonferroni 유의 | 다중비교 통과 | **가장 강한 통계 증거** (Goal 2 만) |
| ⚠️ 비유의 | CI 0 포함 | **유의하지 않음** (alpha 0 일 수도) |
| ⚠️ 단일검정 유의 | CI > 0 but Bonferroni 실패 | **다중비교 후 사라짐** (Goal 2) |
| ❌ 유의(-) | CI 0 미만 | **alpha < 0 통계적으로 입증** |

### **실전 활용**

| **라벨** | **행동** |
| --- | --- |
| ✅ Bonferroni 유의 | paper trading 으로 1주일 검증 후 실거래 |
| ✅ 유의(+) | 더 긴 기간으로 재검증 후 paper trading |
| ⚠️ 비유의 | 데이터 부족. 무시 또는 보강 |
| ❌ 유의(-) | **사용 금지**. SPY 대비 underperform |

새 라벨:
- ci_low > 0 AND p < 0.00208 → "✅ Bonferroni 유의"
- ci_low > 0 (only) → "⚠️ 단일검정 유의 (다중비교 후 비유의)"
- 그외 → "❌ 비유의"

## 1년 vs 5년 비교
이전 ./backtest_results/raw_trades.parquet 와 5년 결과 직접 비교 표:
| skill | 1y alpha | 5y alpha | 차이 | 안정성 (CI 폭) |
해석:
- 차이 ≤2%p: 안정적 alpha
- 차이 >5%p: 1년 결과 unreliable
- 1년만 유의 + 5년 비유의: 강세장 우연 가능성 ↑

## **💬 결과 해석 가이드**

### **Alpha 유의성 라벨 의미**

| **라벨** | **의미** | **해석** |
| --- | --- | --- |
| ✅ 유의(+) | CI 0 초과 | **통계적으로 alpha > 0** (운빨 가능성 낮음) |
| ✅ Bonferroni 유의 | 다중비교 통과 | **가장 강한 통계 증거** (Goal 2 만) |
| ⚠️ 비유의 | CI 0 포함 | **유의하지 않음** (alpha 0 일 수도) |
| ⚠️ 단일검정 유의 | CI > 0 but Bonferroni 실패 | **다중비교 후 사라짐** (Goal 2) |
| ❌ 유의(-) | CI 0 미만 | **alpha < 0 통계적으로 입증** |

### **실전 활용**

| **라벨** | **행동** |
| --- | --- |
| ✅ Bonferroni 유의 | paper trading 으로 1주일 검증 후 실거래 |
| ✅ 유의(+) | 더 긴 기간으로 재검증 후 paper trading |
| ⚠️ 비유의 | 데이터 부족. 무시 또는 보강 |
| ❌ 유의(-) | **사용 금지**. SPY 대비 underperform |
